MRGAZE Gaze Tracking Package
----------------------------

Provides scripts to perform offline pupilometry and gaze estimation from eyetracking video.
- Assumes dark-pupil infrared video
- Requires both a calibration and gaze tracking video for gaze estimation
- Pure python
- Relies heavily on numpy, scipy, opencv

Full documentation can be found at the MRGAZE GitHub repository:

    https://github.com/jmtyszka/mrgaze/wiki
